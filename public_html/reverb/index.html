Convolution Reverb

Description

Web Audio is based on nodes. You can compare it to the way MaxMSP works, but then you make the connection by writing code.

As exercise we well try to combine three (existing) nodes. We will use a sound source, a gain node and a convolution reverb. The UI is already provided (unless you want something else entirely) so you only have to write the webaudio part!
